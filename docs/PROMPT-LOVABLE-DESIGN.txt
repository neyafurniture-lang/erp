MISSION
1) Pose-moi TOUTES les questions de la section 8 du cahier (A à G) avant de designer.

3) Ensuite génère les pages UNE PAR UNE selon `cahier-pages-lovable-une-par-une.md` (prompts de chaque fiche).

QUALITÉ
Niveau SaaS ~20k$ : hiérarchie, respiration, animations intentionnelles (2–3 max/écran).
Mobile = app native. Desktop = shell + contenu.
FR-CA. Accessibilité AA.

IMPORTANT
Réponds d’abord UNIQUEMENT par la liste de questions A–G du cahier, puis attends mes réponses.
Ne design jamais plusieurs pages P0/P1 dans le même message : une fiche = une page.
